# PedeJá - Termos de Uso

Este projeto é uma página web estática que exibe os Termos de Uso do aplicativo PedeJá.

## Estrutura do Projeto

- `index.html`: O arquivo principal contendo o conteúdo HTML.
- `styles.css`: O arquivo de estilos CSS para a formatação da página.

## Como Visualizar

Para visualizar o site, basta abrir o arquivo `index.html` em qualquer navegador web moderno (Chrome, Firefox, Edge, etc.).

Se você estiver usando o VS Code, pode instalar a extensão "Live Server" para visualizar as alterações em tempo real.
